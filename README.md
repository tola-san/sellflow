# SellFlow

> Free & Open Source Digital Product Catalog built with Laravel 12 REST API and Next.js.

---

# 📖 Overview

Sellflow is a modern digital product catalog that helps small businesses showcase products online through a unique URL and QR code.

Customers can browse products without creating an account, while business owners manage everything from a dashboard.

## Project Goals

* Build a production-ready Laravel REST API
* Learn modern Laravel architecture
* Practice API-first development
* Create a portfolio-quality project
* Deploy to production
